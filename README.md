# ssl-mk-notes

## 项目简介

本仓库用于归档和存储多个独立的 HTML 页面。这些页面以静态文件的形式存放，内容主要涉及技术笔记、参考资料及个人整理的信息片段。仓库不针对特定网站或域名，仅作为页面归档与发布的统一容器。

## 目录结构

```
ssl-mk-notes/
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放各独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 公共资源文件（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **pages/**：核心内容目录，每个 HTML 文件代表一个独立的归档页面。
- **assets/**：存放页面所需的公共资源，便于复用和维护。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间相互独立，可通过导航页或直接访问 URL 查看。
- 归档内容可能包含代码片段、笔记、参考资料整理等，不保证时效性或完整性。
- 建议使用现代浏览器访问以获得最佳效果。
